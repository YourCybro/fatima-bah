# 🚀 QUICK START — Fatima Bah Landing Page

## One-Minute Setup

### 1. Open Terminal in This Folder
```bash
cd fatima-bah-landing
```

### 2. Install Everything
```bash
npm install
```

### 3. Start the Site
```bash
npm run dev
```

### 4. Open Your Browser
```
http://localhost:3000
```

**That's it!** 🎉 Your luxury landing page is running.
